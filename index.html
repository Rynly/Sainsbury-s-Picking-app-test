FAQ:

1. How to identify an order?

Each order sent into the Rynly app is unique. This is because every order contains different items and may come from different customer ordering channels. These include Just Eat, Uber Eats and Chop Chop.

It is important to recognise which channel the order belongs to because each channel has slightly different workflows. For example, each channel requires a different process when requesting an additional driver for large orders.
